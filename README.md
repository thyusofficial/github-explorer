## üíª Getting started

### Requirements

- [Node.js](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/)

**Clone the project and access the folder**

```bash
$ git clone https://github.com/thyusofficial/github-explorer.git && cd github-explorer
```

**Then**

```bash
# Install the dependencies
$ yarn

# Start the client
$ yarn start
```
